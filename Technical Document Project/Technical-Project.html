Introduction

Created for CSS (Cascading Style Sheets) user interface desgin, flexbox was created as another way to adjust rows and columns in a more efficient way. This box model design allows to make adjustments to height, width, gap spacing, and other items inside of a container. This works for using nested elements inside a parent element (container) to lay out childern elements in rows or columns. Flexbox is also responsive so it can adjust to different media types.

This document will tech the principle of Flexbox and how to create 'Flex-Direction' and 'Flex-alignment'.

What you should know up to this point

Flexbox is a great tool to shape containers in your webpage on CSS. Having the foundemental understanding of HTML (Hypertext Markup Language) and CSS is needed before one can learn flexbox.

Containers

When using flexbox, it is important to understand the parent-child dynamics.

The Parent will be the main container which contains direct descendance - known as the Child container. Any elements of direct relationship to the parent will have their layouts effected inside the container.
This excludes 'grand-children' - non direct descendance.

Children Items

A child item, or known as child element is what is directly connected to the parent.
To create a child element, there needs to be an element nested inside the parent in HTML.

Flex Direction

FlexBox works by allowing the children items in the container to be established on two different axis.

Flex direction is an part of how child items are placed inside the parent container. There are multiple different properties that effect the direction of the flex box itself.

in order to establish the flex-direction the parent container needs to have the property set to Flex-Direction while the values will establish the direction of the flexbox items.

Values: Column | Row | Column-Reverse | Row-Reverse
