Introduction

Flexbox was created as another way to adjust rows and columns in a more efficient way. Being able to adjust the height, width, gap spacing, and other items inside of a container.
Parent Container

When using flex, it is important to understand the parent-child dynamics.

The Parent will be the main container which contains direct descendance - known as childern. Any elements of direct relationship to the parent will have their layouts effected inside the container.
This excludes 'grand-children' - non direct descendance.

Child Container
To start using flex, the parent container needs to be changed to [display:flex;].

Children Items

A child item, or known as child element is what is directly connected to the parent.
To create a child element, there needs to be an element nested inside the parent in HTML.

Flex Direction

FlexBox works by allowing the childern itmes in the container to be established on two different axis.
Flex direction is an part of how child items are placed inside the parent container. There are mutiple different Properties that effect the direction of the flex box it self.

The main flex directions

  • Column
  • Column-Reverse
  • Row
